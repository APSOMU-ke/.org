# APSOMU Website

The official website for the Association of Political Science Students of Moi University (APSOMU).

## Project Overview

APSOMU is a student-led organization dedicated to empowering Political Science scholars through academic enrichment, professional development, and community engagement. This website serves as a central hub for members to learn about our mission, stay updated on events, and connect with the community.

## Features

- **About Us**: Information about the association's history and leadership.
- **Goals & Impacts**: Insights into our mission and the impact we're making.
- **Events**: A timeline of past and upcoming association activities.
- **Stakeholders**: An interactive overview of the key partners and stakeholders.
- **Gallery**: A visual showcase of our journey.

## Technologies

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Getting Started

To view the website locally:
1. Clone the repository.
2. Open `index.html` in your web browser.
